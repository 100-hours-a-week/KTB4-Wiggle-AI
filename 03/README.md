## 회고
numpy, pandas와 matplotlib, seaborn은 정말 많이 사용해봐서 큰 어려움 없이 위클리 챌린지를 진행할 수 있었다.
그렇다고 복습 수준에서 나아가 더 얻은 게 없지는 않았다. pandas나 시각화 라이브러리처럼 자주 사용되는 라이브러리일수록 자주 사용하는 기능만 관성적으로 계속 쓰게 되는 경향이 있는 것 같다. 내게는 dataframe에서 문자열을 다루는 상황이 그랬는데, dataframe.str에 내장된 메서드들을 사용하지 않고 파이썬 기본 문자열 함수나 정규식을 사용해 전처리를 진행하는 경우가 많았다. 이번 기회에 pandas 기능들을 처음부터 훑어보며 내가 관성적으로 사용하던 기능들보다 더 효율적이고 깔끔한 방법들이 있다는 걸 알 수 있었다.
